# Privacy Policy

Last updated: 2026-06-01

Kindle Series Sale Tracker is a browser extension for organizing a user's Amazon.co.jp Kindle library into series candidates and follow-up volume checks.

## Data handled by the extension

The extension runs only on Amazon.co.jp Kindle library pages and Amazon.co.jp search result pages used by the extension features. When the user starts a scan, it reads Kindle library item data available in the user's browser session, such as book titles, authors, ASINs, read status, acquisition timestamps, and cover image URLs. It also reads Amazon search result page content when the user checks follow-up volume candidates, such as candidate titles, URLs, prices, discounts, release dates, and thumbnails.

## How data is used

The data is used only to provide the extension's user-facing features:

- Group owned Kindle books into series candidates.
- Show owned volume ranges and missing volume estimates.
- Check possible follow-up volumes and display price or discount information when available.
- Export the user's local scan result as CSV or JSON at the user's request.

## Storage and sharing

The extension stores scan results and settings locally in browser extension storage. It does not send Kindle library data, browsing data, exported files, or settings to the developer or to any developer-operated server.

The extension communicates only with Amazon.co.jp pages that the user is already using, in order to fetch Kindle library data and search result pages needed for the extension features.

## Authentication and cookies

The extension does not collect, store, or transmit Amazon passwords, cookies, or login credentials. Requests to Amazon.co.jp are made from the user's browser session so that Amazon can authorize the user's own library page.

## Data deletion

Users can clear stored scan results and cached follow-up checks from the extension UI. Users can also remove all extension data by uninstalling the extension or clearing the extension's storage from the browser.

## Limited use

The extension's use of user data is limited to providing and improving its single purpose: organizing a user's Amazon.co.jp Kindle library and follow-up volume checks locally in the browser. The extension does not sell user data, use user data for advertising, or share user data with third parties.
